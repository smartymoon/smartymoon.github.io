python 包记录

开始入坑 python，记录下用过的包，以便日后查询

  1. docopt python 命令行参数解释器，利用 __doc__ 实现
  2. tkinter 内置 GUI 模块，用于简单需求
  3. xlrd Excel 读取工具，可以读 xls,xlsx 不能打开被保护的文件
  4. glob 读取目录下的文件，可以使用通配符
  5. os.path 目录操作，可以判断是否目录
  6. threading 多线程，python 多线程是坑，可以用来做批量延时任务
  7. requests 非常出名的网络请求工具，用法和 Axios 很像
  8. re 正则，可匹配，可搜索
  9. pyquery 类似Jquery的 Dom 选择器
